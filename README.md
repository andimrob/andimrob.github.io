Personal portfolio site built with React, TypeScript, Tailwind CSS, and Vite.

## Prerequisites

- Node.js 20+
- npm

## Local Development

```bash
pnpm install
pnpm run dev
```

## Build

```bash
pnpm run build
```

Output goes to the `public/` directory. Preview the production build locally:

```bash
pnpm run preview
```

## Deployment

Pushes to `main` automatically build and deploy to GitHub Pages via the workflow in `.github/workflows/main.yml`.

## Easter Eggs

- Press **X** to toggle an x-ray lens that reveals the site's syntax-highlighted minified source code
- Click the **navbar** to flip it and see random quips
- Check the **browser console** for ASCII art and hints
